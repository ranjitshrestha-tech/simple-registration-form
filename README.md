This project helps me improve the basics of UI design, form structure, and responsive layout.

Features-
Simple and clean UI,
Email and password fields,
Basic form structure ,and
Beginner-friendly code.

Tech Used-
HTML,
CSS.

Clone the repo,
Open index.html in any browser ,and
Edit and customize the form as you like.
